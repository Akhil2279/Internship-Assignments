>> Simple Expression Calculator CLI

A beginner-friendly Python CLI calculator that evaluates **full mathematical expressions**.  
You can type multiple operators at once, including parentheses, and see the result instantly.  


>> Features
- Supports arithmetic operations: `+`, `-`, `*`, `/`
- Parentheses `()` for controlling order of operations
- Handles division by zero
- Exit the program with `exit`
- Minimal, beginner-friendly design


>> Requirements
- Python 3.8 or higher installed on your system


>> How to Run

1. Save the code into a file named `calculator.py`.
2. Open a terminal or command prompt in the folder where the file is saved.
3. Run:
   ```bash
   python calculator.py
