>> Simple To-Do List App

A beginner-friendly Python command-line To-Do list application.  
This app allows you to **view, add, and remove tasks** and saves them in a file (`todo.txt`) so your tasks persist between sessions.

---

>>  Features
- View all your tasks
- Add new tasks
- Remove tasks by number
- Automatically save tasks to a file
- Easy-to-use command-line interface

---

>> Requirements
- Python 3.6 or higher installed on your system

---

>> How to Run

1. Save the code into a file named `todo.py`.
2. Open your terminal or command prompt in the folder where the file is saved.
3. Run:
   ```bash
   python todo.py
