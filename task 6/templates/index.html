View Project

Backend API Developement

I worked on developing RESTful APIs for the KPA Form project using FastAPI and PostgreSQL, ensuring the endpoints aligned with the SwaggerHub documentation and were ready to integrate with a Flutter frontend. The implementation included a GET API for fetching Wheel Specifications and a POST API for submitting Bogie Checksheet data. The backend was structured with models (SQLAlchemy), schemas (Pydantic for validation), routes, and a main application entry point. I also upgraded the codebase to be compatible with Pydantic v2.x, resolving ForwardRef evaluation issues. The project emphasized clean architecture, proper data validation, error handling, and JSON-based responses, and I thoroughly tested the APIs using Postman to ensure reliability and accuracy.
